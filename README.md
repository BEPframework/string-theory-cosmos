# String Theory Cosmos

**v1.1.0 — Research Instrument Edition**

An interactive, fully deterministic, browser-based toy model for exploring string/M-theory and modern cosmology concepts in a single self-contained HTML file.

### Features
- Deterministic evolution using integer seed (reproducible runs)
- Real-time visualization of open/closed strings, D-branes, Calabi-Yau projection, wormholes, eternal inflation bubbles, and evaporating black holes
- 1 Hz time-series logging of observables
- Single-run export (JSON + CSV)
- **New in v1.1**: Full headless parameter sweep mode (scan any parameter across a range, multiple runs per point, averaged results, progress bar)
- Sweep export (JSON + CSV) for further analysis
- Keyboard shortcuts and clean UI

### How to use
1. Open `string-theory-cosmos.html` in any modern browser (no installation needed).
2. Play with sliders, click to add strings, double-click to reseed.
3. Use **Pause** to see live observables.
4. Use **Parameter Sweep** (in the UI) for systematic scans.
5. Export data for Python/Mathematica analysis.
